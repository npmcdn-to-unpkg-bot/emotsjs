# Instalation  


``` 
 npm install --save emotsjs
```


# how to use


```javascript

	var emotjs = require('emotsjs');
    
    emotjs.all; // return all emots
    emotjs.random(); // return a random emot
    emotjs.random(n); // return n emots random
 	emotjs.get('rocket'); // return 'rocket' emot
 ```
